# ReactBank
# Gestión de Turnos en Banco

## Descripción
La página está orientada a un gestor de turnos bancarios, donde, una vez iniciada la sesión, se pueden solicitar nuevos turnos, ver los turnos anteriores y también cancelarlos.

## Tecnologías
### Backend:
- Node.js
- Express
- TypeScript
- PostgreSQL

### Frontend:
- React.js + Vite
- JavaScript
- CSS
